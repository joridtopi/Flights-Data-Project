Dependencies:

The local web servers requires Python 2.x or Python 3.x to be installed in your local machine




In order to access the web based visualizations, perform the following steps:

1. In the operating system command terminal, change the directory to the following project location:

~/Flights_Data_Project/Visualization/D3_Vis

2. This will activate the local web server on port 8888. If running Python 2.x, execute the following command in the terminal:

"python -m SimpleHTTPServer 8888 &."

If running Python 3.x, execute the following command in the terminal:

"python -m http.server 8888 &."

3. In a web browser, visit the following URL to access the home page of the project:

"http://localhost:8888/"
