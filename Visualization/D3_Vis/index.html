About:

This page explores the domestic flight data recorded by the Bureau of Transportation Statistics with visual charts. The data of all the flights from the years 1988 to 2019 is stored in a database from where relevant features of interest are extracted using aggregation functions and groupings.

Features For Flight Performance:

Number of Departed Flights

Average Departure Delay

Number of Cancelled Flights

Percentage of Cancelled Flights

Features For Flight Planning:

Number of Scheduled Flights

Number of Flight Routes

Number of Aircraft Used
