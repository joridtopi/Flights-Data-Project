About:

This page explores visualizations of open source domestic flight data recorded by the Bureau of Transportation Statistics (BTS) with web based charts. The data of all the flights from the years 1988 to 2019 is stored in a database from which relevant features of interest are extracted using aggregation functions and groupings. The open source data page from the BTS can be accessed via this link.

Flight Performance Metrics:

  • Number of Departed Flights
  • Average Departure Delay
  • Number of Cancelled Flights
  • Percentage of Cancelled Flights
  • Number of Miles Flown

Flight Planning Metrics:
  • Number of Scheduled Flights
  • Number of Flight Routes
  • Number of Aircraft Used
